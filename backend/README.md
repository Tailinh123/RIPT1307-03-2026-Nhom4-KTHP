# Intern Matching - RESTful API

## Overview

Intern Matching is a RESTful API built with Java Spring Boot. This system is designed to provide a platform connecting university students seeking internships with employers.

## Features (Ongoing)

*Note: This project is currently under active development. Core modules are being built step-by-step.*

- **User Authentication**: Secure login and registration (In progress).
- **Core Entities Management**: Structuring databases for Students, Companies, Jobs, and Majors.
- **REST API**: Standardized JSON responses and global exception handling.

## Tech Stack

- **Java**: Version 17
- **Framework**: Spring Boot 3.3.2
- **Build Tool**: Gradle
- **Database**: MySQL 8.0
- **ORM**: Spring Data JPA
